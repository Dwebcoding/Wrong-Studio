# Wrong Studio - Portfolio di Architettura

Un sito web portfolio contemporaneo per architettura e design d'interni che presenta progetti minimalisti e brutalisti.

## Panoramica del Progetto

Wrong Studio è un sito web portfolio moderno e responsive per uno studio di architettura e design d'interni. Presenta un design pulito e minimalista con animazioni fluide e un'interfaccia utente elegante.

## Struttura del Progetto

```
.
├── index.html              # Pagina principale
├── pages/                  # Pagine aggiuntive
│   ├── works.html         # Pagina portfolio/lavori
│   ├── about.html         # Pagina chi siamo
│   └── contact.html       # Pagina contatti
├── css/                   # Fogli di stile
│   └── style.css          # Stili principali
├── js/                    # File JavaScript
│   └── script.js          # Script principale
├── assets/                # Immagini e media
│   ├── logo/              # File logo
│   └── images/            # Immagini progetti
├── .gitignore             # File ignore Git
└── README.md              # Questo file
```

## Pagine

- **Home (index.html)** - Sezione hero con lavori in evidenza
- **Works** - Portfolio completo dei progetti
- **About** - Informazioni studio e team
- **Contact** - Form contatti e informazioni

## Caratteristiche Design

- Design responsive (approccio mobile-first)
- Animazioni e transizioni fluide
- Tipografia moderna con font Inter
- Estetica minimalista ad alto contrasto
- Struttura HTML ottimizzata per SEO

## Come Iniziare

1. Clona la repository
   ```bash
   git clone https://github.com/yourusername/wrong-studio.git
   ```

2. Naviga nel progetto
   ```bash
   cd wrong-studio
   ```

3. Apri `index.html` nel tuo browser
